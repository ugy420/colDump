<ul>
Represents an unordered lost of items, typically rendered as a bulleted list
<ol>
Represents an ordered list of items, typically rendered as a numbered list.
<li>
used to represent an item in a list. It must be contained in a parent element:
an ordered list (<ol>), or an unordered list (<ul>).
<dl>
Represents a description list.
The element encloses a list of groups of terms and descriptions.
<dt>
Identifies a term in a description list. This element can occur only as a child
element of a <dl>. It is usually followed by a <dd> element.
<dd>
Indicates the description of a term in a decription list(<dl>).

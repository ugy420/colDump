HTML
HTML stands for Hypertext Markup Language. It is a standard markup language used for creating web pages and applications.
HTML Element
An HTML element is a building block of an HTML page. It is defined by a starting tag, some content, and an ending tag.
HTML attribute
HTML attributes provide additional information about an HTML element. Attributes are defined in the starting tag of an HTML element and consist of a name and a value.
HTML comment
An HTML comment is a piece of text that is not displayed on the web page but is used to provide information about the HTML code.
XML
XML stands for Extensible Markup Language. It is a markup language used for storing and transporting data.
CSS
CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the presentation of HTML or XML documents.
JS
Javascript is a high-level, interpretted programming language that can be used for a variety of purposes, including web development, server-side scripting, and game development.
